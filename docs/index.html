✨ Características

Todo lo que necesitas para generar contribuciones profesionales.

🎯

Patrones Realistas

Distribución ponderada de commits que simula actividad humana natural con variaciones de frecuencia e intensidad.

⏰

Ejecución Programada

GitHub Actions ejecuta el bot automáticamente cada día a medianoche UTC con soporte para trigger manual.

🔧

Altamente Configurable

Personaliza fechas, frecuencia, horarios, probabilidad de fines de semana y más a través de config.json o CLI.

📝

Conventional Commits

Mensajes de commit profesionales siguiendo el estándar Conventional Commits: feat, fix, refactor, docs, y más.

🔍

Modo Dry-Run

Previsualiza exactamente qué commits se generarán sin hacer ningún cambio real al repositorio.

🎨

CLI Profesional

Interfaz de terminal con colores ANSI, barra de progreso en tiempo real y resumen detallado al finalizar.

🚀 Quick Start

Comienza a generar contribuciones en 3 pasos.

terminal 
# 1. Clona el repositorio
git clone https://github.com/YOUR_USER/Auto-Commits.git
cd Auto-Commits

# 2. Previsualiza los commits (modo seguro)
python scripts/bot.py --dry-run

# 3. Genera commits reales
python scripts/bot.py

# Opciones avanzadas
python scripts/bot.py --days 180 --max-commits 5 --no-push
